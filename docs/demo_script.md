# Five-minute demo script

## 0:00 - Context

Present SocialCare Subsidy Agent as a portfolio and hackathon prototype inspired by social-impact work, using simulated data only.

## 0:45 - Pain point

Explain that vulnerable users often do not know which channel to use, which documents to prepare, or when their case needs human support.

## 1:30 - Case 1: unemployment subsidy

Ask: "Perdí mi empleo y quiero saber qué documentos generales podría revisar."

Show classification, local RAG response, clear limits, and no approval or denial.

## 2:30 - Case 2: child services

Ask: "Necesito orientación para servicios de mi hijo pequeño."

Show the child-services category and general safe guidance.

## 3:30 - Case 3: sensitive escalation

Ask: "Hay posible maltrato a un menor y necesito ayuda urgente."

Show high risk, escalation requirement, and safe advisor summary.

## 4:30 - Closing

Close with technical capabilities: AI agents, RAG, Responsible AI, FastAPI, Streamlit, DevOps, observability, Azure readiness, and human-centered escalation.
