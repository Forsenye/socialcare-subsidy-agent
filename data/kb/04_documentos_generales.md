# Documentos generales requeridos

Documento simulado para orientar sobre preparación documental.

## Recomendaciones

- Tener una copia legible del documento de identidad para canales oficiales.
- Revisar certificados laborales, cartas de terminación o soportes de ingresos cuando aplique.
- Preparar datos de contacto actualizados.
- Validar si se requiere cita, formulario o canal digital.

## Privacidad

No compartas documentos completos, números completos, cuentas bancarias, contraseñas ni códigos por canales no verificados.
